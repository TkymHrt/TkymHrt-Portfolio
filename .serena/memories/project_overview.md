# TkymHrt Portfolio - プロジェクト概要

## プロジェクトの目的
TkymHrtのポートフォリオサイト。ユーザー向けのポートフォリオページに加えて、内容を修正・追加できる管理者ページを提供している。

## 技術スタック

### フロントエンド
- **Next.js 14.2.8** (App Router)
- **React 18** 
- **TypeScript 5**
- **Tailwind CSS 3.4.1**
- **Framer Motion 11.5.4** (アニメーション)

### UI/UX Enhancement
- **@tsparticles/slim** (パーティクル効果)
- **react-scroll** (スムーズスクロール)
- **@vercel/analytics** (分析)

### 開発ツール
- **Biome 1.8.3** (リンター・フォーマッター)
- **PostCSS**
- **@cloudflare/next-on-pages** (Cloudflare Pages対応)

## デプロイメント
- **Vercel**: https://tkymhrt-portfolio.vercel.app/
- **Netlify**: https://tkymhrt-portfolio.netlify.app/

## プロジェクト特徴
- レスポンシブデザイン
- パーティクル効果を使用したインタラクティブなUI
- Framer Motionによるスムーズなアニメーション
- 管理者ページ機能
- 複数プラットフォームでのデプロイ対応
