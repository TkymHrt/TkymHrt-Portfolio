<h2>🛠️ Installation and Running</h2>

<p>1. ファーストコマンド</p>

```
make build
```

<p>2. アプリ立ち上げ</p>

```
make run
```

<p>アプリ再起動</p>

```
make rebuild
```

<p>prisma studio 立ち上げ</p>

```
make studio
```

<p>DB のリセット</p>

```
make dbreset
```

<p>seedの適用</p>

```
make seed
```
